# Bumblebeeschool
***The web application is an educational platform for the online English language school (Bumblebeeschool).***

## Main tools:
```
- Python 3.10.2
- Django 4.0.3
- Django-ckeditor 6.2.0
- Django-taggit 2.1.0
- Pillow 9.1.0
- Bootstrap 5.0
- HTML 5
- CSS 2.1
```
